# Yatube API

Yatube API — это полнофункциональный REST API для социальной сети, разработанный на Django и Django REST Framework. Проект предоставляет программный интерфейс для управления публикациями, комментариями, подписками и сообществами.

## Основные возможности
- Аутентификация и авторизация пользователей

Приложение использует JWT-токены для безопасной аутентификации пользователей. Реализована система получения, обновления и проверки токенов через эндпоинты /api/v1/jwt/create/, /api/v1/jwt/refresh/ и /api/v1/jwt/verify/. Разграничение прав доступа: анонимные пользователи имеют доступ только на чтение, аутентифицированные пользователи могут создавать и редактировать свой контент.

- Работа с публикациями

API поддерживает полный CRUD для постов: создание, чтение, обновление и удаление. Каждая публикация может содержать текст, изображение и принадлежать к сообществу. Реализована пагинация с параметрами limit и offset. Только автор публикации может редактировать или удалять её — при попытке изменения чужого контента возвращается ошибка 403.

- Работа с комментариями

В приложении реализована вложенная система комментариев: каждый пост может иметь неограниченное количество комментариев. Авторизованные пользователи могут создавать, редактировать и удалять свои комментарии.

- Подписки на пользователей

Реализован механизм подписок: пользователи могут подписываться на других авторов. Эндпоинт /api/v1/follow/ поддерживает получение списка своих подписок и поиск по имени пользователя через параметр search. Предусмотрена защита от подписки на самого себя и дублирования подписок.

## Настройка и запуск
1) Клонируйте репозиторий и перейдите в него в командной строке:
   ```bash
   git clone https://github.com/LE1-Norman/api-final-yatube-ad.git
   cd api-final-yatube-ad
   ```
2) Создайте и активируйте виртуальное окружение
   ```bash
   python -m venv venv
   source venv/Scripts/activate  # Windows (Git Bash)
   ```

3) Установите зависимости
   ```bash
   pip install -r requirements.txt
   ```
4) Выполните миграции
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```
5) Запустите проект
   ```bash
   python manage.py runserver
   ```
6) После запуска документация API будет доступна по адресу: http://127.0.0.1:8000/redoc/
   
7) Примеры различных запросов к API можно найти в документации
